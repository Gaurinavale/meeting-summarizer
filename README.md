# 🎙️ AI Meeting Summarizer

AI-powered meeting summarizer that transcribes audio/video and generates structured summaries.

## 🚀 Features
- 🎙️ Transcribe MP3, WAV, MP4, TXT using Whisper
- 🤖 AI summarization via OpenRouter
- ✅ Action items with owners and deadlines
- 📊 Structured JSON export
- 🌐 Streamlit web UI

## ⚙️ Setup
1. Clone the repo
2. Run `pip install -r requirements.txt`
3. Create `.env` with your `OPENROUTER_API_KEY`
4. Run `streamlit run app.py`

## 🛠️ Tech Stack
Python • Whisper • OpenRouter • Streamlit • MoviePy
